Content Security Policy (CSP) demo

 

Open the Console tab on Developer Tools to see CSP violations.

 

 

Stylesheets

The Bootstrap styles & fonts should load because bootstrapcdn.com and *.bootstrapcdn.com are whitelisted on the style-src and font-src directives.

 

 

Scripts

The jQuery & Bootstrap scripts should load because bootstrapcdn.com, *.bootstrapcdn.com, googleapis.com, and *.googleapis.com are whitelisted on the style-src and font-src directives.

 

 

Images

The image below should load because imgur.com and *.imgur.com are whitelisted on the img-src directive.

 

The image below should NOT load because *.9cache.com is not whitelisted on the img-src directive.

 

 

Forms

The form below should submit because 'self' is whitelisted on the form-action directive.

 

The form below should NOT submit because *.google.com is not whitelisted on the form-action directive.

 

 

IFrames

This page should not be embeddable on other sources because 'none' is the value specified on the frame-ancestors directive.

 

 

AJAX

This button should get an API response because pokeapi.co is whitelisted the connect-src directive.

 

This button should not get an API response because swapi.co is not whitelisted on the connect-src directive.
